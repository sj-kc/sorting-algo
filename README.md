## A VanillaJS visualization tool for Sorting Algorithms

### Clone the repo

```bash
git clone https://github.com/sungkimcastro/sorting-algo.git
```

### Install dependencies

```bash
yarn install
```

### Start a local server to preview the site

```bash
yarn dev-server
```
